## Development Setup

```bash
yarn install
```

### Build

```bash
# all packages
yarn build
# @vue packages
yarn build:vue-next

# only one package and watch
# modify the `PACKAGE_PATH` and `target` in `scripts/dev.js` first, and then
yarn dev
```

### serve

```bash
# open browser
yarn serve
```
